# 🎮 Atrapar Objetos

Un juego casual donde el jugador debe atrapar pelotas y evitar obstáculos. ¡Desarrollado en Visual Studio!

## Cómo jugar
- Usa las flechas del teclado para moverte.
- Atrapa pelotas para ganar puntos.
- Evita las bombas para no perder vidas.

## Recursos
- `Form1.cs`: ventana principal del juego.
- `Acciones.cs`: lógica de movimiento y colisiones.
- `Dificultad.cs`: niveles de dificultad.
